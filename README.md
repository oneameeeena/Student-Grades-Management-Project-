# Student-Grades-Management-Project-

Overview

This Python program allows you to manage a classroom’s student records and grades. It provides features to add, remove, and update students, manage grades by subject, generate class reports, and calculate averages. The program includes an interactive menu for easy testing and usage.

Features
Student Management

Add a new student with an ID and name.

Remove a student by ID.

Display details of a specific student.

Display all students and their grades.

Search for a student by name.

Grade Management

Add or update a student’s grade for a specific subject.

Remove a student’s grade for a specific subject.

Class Report Management

Add a student’s grades to the class report.

Remove a student’s grades from the class report.

Display a student’s report.

Display all student reports.

Calculate a student’s total average grade.
